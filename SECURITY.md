# Security Policy

## Supported Versions

Security updates are provided for the latest released version of this module.

| Version | Supported |
| ------- | --------- |
| Latest  | :white_check_mark: |
| Older   | :x: |

## Reporting a Vulnerability

Please report security issues privately and do not open a public GitHub issue.

Send a report to: security@siteation.dev

Include the following details when possible:
- A clear description of the issue
- Steps to reproduce
- Affected module version
- Potential impact
- Any suggested remediation

You will receive an acknowledgement within 3 business days. After confirmation, we will work on a fix and coordinate a responsible disclosure timeline.
